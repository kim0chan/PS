## 21940 : 가운데에서 만나기
플로이드 응용 문제  

거리 배열을 채울 때 2차원 배열을 `fill()` 함수로 채우는 테크닉
```cpp
for (int i = 1; i <= N; i++) {
	fill(dist[i], dist[i] + N + 1, INF);
	dist[i][i] = 0;
}
```
생각 안나면 클래식하게
```cpp
for (int i = 1; i <= N; i++) {
	for (int j = 1; j <= N; j++) {
		if (i == j)	continue;
		dist[i][j] = INF;
	}
}
```
