## 1181 : 단어 정렬
문자열, 구현  
문자열의 비교 함수 `compare`에 관하여  
`string a, string b`이 있을 때 `a.compare(b)`와 같이 사용하고,  
`b`가 사전순으로 더 크면 `-1`, 같으면 `0`, 작으면 `1`을 반환한다.  
(a - b)를 한 결과라고 생각하자.

## 1182 : 부분수열의 합
브루트포스, 백트래킹  
('23. 9. 11.) 합이 `0`이 되는 경우는 전체 부분 수열 가짓수에서 `1`을 빼주어야 한다.  
크기가 양의 정수가 아니고 `0`인 부분 수열이 존재하기 때문이다.
```cpp
if (s == 0) cnt--;
cout << cnt << '\n';
```
