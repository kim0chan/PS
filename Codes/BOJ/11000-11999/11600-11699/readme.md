## 11653 : 소인수분해
소인수분해 알고리즘 잘 알아두자.  
```cpp
for (int i = 2; i * i <= N; i++) {
	while (N % i == 0) {
        cout << i << '\n';
		N /= i;
	}
}

if (N != 1) cout << N << '\n';
```
