## 2609 : 최대공약수와 최소공배수
최대공약수 알고리즘(유클리드 호제법)
```cpp
int gcd(int a, int b) {
    if(a == 0)  return b;
    return gcd(b % a, a);
}
```
최소공배수 = $GCD(A, B) \times LCM(A, B)$
